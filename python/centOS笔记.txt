--ls常用选项
-a 显示指定目录下所有目录与文件，包括隐藏文件
-l 以列表方式显示文件的详细信息
-h 配合-l以人性化的方式显示文件大小


--查看文件内容
1.cat 文件名 concatenate 查看文件内容，创建文件，文件合并，追加文件内容等功能

2.more 文件名 more 分屏显示文件内容

3.grep 搜索文本文件名 grep 搜索文本文件内容


--grep
1.Linux系统中grep命令时一种强大的文本搜索工具

2.grep允许对文本文件进行模式查找，所谓模式查找，又称为正则表达式

3.-n 显示匹配行及行号

4.-v显示不包含匹配文本的所有行（相当于求反）

5.忽略大小写

6.常用的两种模式查找
^a 行首，搜寻以a开头的行
ke$ 行尾，搜寻以ke结束的行


--echo文字内容
echo会在终端中显示参数指定的文字，通常会和重定向联合使用


--重定向>和>>
1.Linux允许将命令执行结果重定向到一个文件

2.将本应该显示在终端上的内容输出/追加到指定文件中
其中
	>表示输出，会覆盖文件原有的内容
	>>表示最佳，会将内容追加到已有文件的末尾

eg:echo Hello Python > a
这条命令会创建一个名为“a”的文件并将Hello Python写入到a文件中
如果在a文件已经有内容的情况下，再次输入 echo hello python > a,原本a文件中的“Hello Python”会被重写掉，变为“hello python”

如果想在原有文件的基础上添加内容，应使用“>>”
比如：tree >> a
会将当前路径下的文件树输入到a文件中
